# Photos

Put the listing photos in this folder and name them so they match the list in
`assets/app.js`:

```
photo-1.jpg
photo-2.jpg
photo-3.jpg
photo-4.jpg
photo-5.jpg
photo-6.jpg
```

- The first photo (`photo-1.jpg`) is also used as the social/Open Graph preview image.
- Any number of photos works — just keep the filenames in sync with the `PHOTOS`
  array in `assets/app.js`.
- Until real files are added, the gallery shows a styled placeholder for each slot.
- Recommended: landscape orientation, roughly 1600×1200 px, JPG/WebP, < 500 KB each.
